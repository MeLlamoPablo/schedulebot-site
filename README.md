# schedulebot-site

The homepage for [ScheduleBot](https://github.com/MeLlamoPablo/schedulebot/tree/dota).

To build this site:

```sh
$ git clone https://github.com/MeLlamoPablo/schedulebot-site.git
$ cd schedulebot-site
$ npm install
$ npm run build
```

You will find the site contents in the `/dist` folder.
